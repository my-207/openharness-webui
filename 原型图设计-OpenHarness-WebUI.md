## 1. 设计系统

### 1.1 色彩系统

### 1.2 排版

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 28px | Bold (700) | 页面标题 |
| H2 | 22px | Semibold (600) | 区块标题 |
| H3 | 18px | Semibold (600) | 卡片标题 |
| Body | 14px | Regular (400) | 正文文本 |
| Body-Small | 12px | Regular (400) | 辅助文本 |
| Code | 13px | Monospace | 代码/技术内容 |
| Caption | 11px | Regular (400) | 标签/提示 |

### 1.3 间距

| 名称 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 紧凑元素间距 |
| sm | 8px | 元素内部间距 |
| md | 12px | 元素间距 |
| lg | 16px | 区块间距 |
| xl | 24px | 大区块间距 |
| 2xl | 32px | 页面间距 |

### 1.4 圆角

| 名称 | 值 | 用途 |
|------|-----|------|
| sm | 4px | 小元素（标签） |
| md | 6px | 常规卡片 |
| lg | 8px | 大卡片/面板 |
| xl | 12px | 模态框 |

### 1.5 阴影

```css
shadow-sm: 0 1px 2px rgba(0,0,0,0.3)
shadow-md: 0 4px 6px rgba(0,0,0,0.2)
shadow-lg: 0 10px 15px rgba(0,0,0,0.3)
```

### 2.2 导航栏 (NavBar)

```
┌─────────────────────────────────────────────────────────────────┐
│  [≡]  [⚡ OpenHarness]     [🔍 搜索会话/命令...]   [🤖 sonnet ▾] [⚙️] [👤] │
└─────────────────────────────────────────────────────────────────┘
```

| 元素 | 说明 | 交互 |
|------|------|------|
| ≡ (汉堡菜单) | 切换左侧栏显示/隐藏 | 点击切换 |
| ⚡ OpenHarness (Logo) | 应用标识 + 名称 | 点击回到首页 |
| 🔍 搜索框 | 搜索会话/命令 | 输入搜索，实时过滤 |
| 🤖 模型选择器 | 显示当前模型，下拉切换 | 点击展开模型列表 |
| ⚙️ 设置按钮 | 打开设置面板 | 点击跳转 `/settings` |
| 👤 用户头像 | 用户信息/登录 | 点击展开用户菜单 |

### 2.3 左侧栏 (SideBar)

```
桌面端(展开，150px):             移动端(收起):
┌────────────┐                  [≡] → 抽屉弹出
│ [+ 新建]    │
│             │
│ 💬 会话     │ (当前选中，高亮)
│ 🏠 历史     │
│ ⚡ 工具     │
│ 🧠 技能     │
│ 🔌 插件     │
│ 📋 任务     │
│ 🤝 多代理   │
│ ⏰ 定时任务  │
│ 🤖 Autopilot │
│ ⚙️ 设置     │
│             │
│ 🏠 ohmo     │
└────────────┘
```

每个导航项包含：
- 图标 + 名称
- 选中态: 左侧蓝色竖条 + 背景高亮
- hover 态: 背景色变浅
- 徽章（可选）：显示计数

### 2.4 状态栏 (StatusBar)

```
┌─────────────────────────────────────────────────────────────────┐
│  [●] anthropic    [● MCP: 3 connected]    [🤖 claude-sonnet-4]  [⚡ 1,234 tokens] │
└─────────────────────────────────────────────────────────────────┘
```

| 元素 | 颜色 | 含义 |
|------|------|------|
| ● 提供者状态 | 绿/黄/红 | 就绪/警告/错误 |
| ● MCP 状态 | 绿/灰 | 已连接/未连接 + 数量 |
| 🤖 模型 | 信息 | 当前模型名称 |
| ⚡ Token | 信息 | 当前会话 Token 用量 |

## 4. 交互细节说明

### 4.1 流式对话交互

```
1. 用户输入消息 → 输入框动态显示 "..."
2. 发送后 → 用户消息出现在消息列表，输入框变为 "⏳ 等待响应..."
3. AI 开始回复 → 消息列表出现新的 AI 气泡
   - 文本逐步显示（流式效果）
   - 代码块先显示框架，再逐步填充
4. 工具调用开始 → 对应工具卡片弹出，显示 "🔧 ToolName [● 执行中]"
5. 工具调用完成 → 卡片变为 "✅ completed in XXms"，可展开
6. 重复 3-5 直到 AI 回复完成
7. 输入框恢复可用状态
```

### 4.3 斜杠命令交互

```
1. 用户在输入框中输入 "/"
2. 输入框上方出现命令补全下拉列表（最多 10 项）
3. 继续输入过滤命令列表
4. 使用 ↑↓ 键导航选择
5. Enter 选择并执行
6. Tab 补全当前选中命令但不执行
7. Esc 关闭补下列表
```

## 6. 用户流程

### 6.1 首次使用流程

```
1. 打开 WebUI → 欢迎页面
2. 点击 "开始设置" → 跳转提供者配置页
3. 选择提供者工作流 (Anthropic/OpenAI/Copilot)
4. 输入 API Key 或跳转 OAuth 认证
5. 选择模型
6. 回到对话页 → 可以开始聊天
```

### 6.2 日常使用流程

```
1. 打开 WebUI → 进入对话页
2. 查看是否有未完成的对话
3. 继续会话或新建
4. 输入消息 → AI 回复
5. 查看工具调用详情
6. 使用斜杠命令切换模型/权限模式
7. 查看 Token 费用
8. 导出重要对话
```

### 6.3 高级管理流程

```
1. 进入工具管理页 → 查看所有可用工具
2. 进入 MCP 面板 → 添加新的 MCP 服务器
3. 进入定时任务页 → 创建 Cron 作业
4. 进入多代理面板 → 查看队友状态
5. 进入 ohmo 管理 → 配置网关频道
```
